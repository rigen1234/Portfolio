Projects

Home Security System

This project implements a basic home security system using an Arduino microcontroller. It features motion detection via PIR sensors, door intrusion alerts using magnetic reed switches, and real-time notifications through a buzzer or LED indicators. Optional enhancements include GSM modules for SMS alerts and keypad-based access control.

Text to Audio Converter

The main objective of this project is to design and develop a Text to Audio Converter web application that reads aloud any text entered or uploaded by the user. It enhances accessibility for visually impaired users and assists learners who prefer audio-based content.
